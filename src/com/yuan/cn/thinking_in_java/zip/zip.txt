java I/O类库中的类支持读写压缩格式的数据流。你可以用它们对其他的I/O库类进行封装，以提供压缩功能。
    这些类不是从Reader和Writer类派生而来的，而是属于InputStream和OutputStream继承层次结构的一部分。这样做是因为压缩类库是按字节方式而不是字符方式处理的。

    CheckedInputStream      GetCheckSum()为任何inputStream 产生校验和（不仅是解压缩）
    CheckedOutputStream     GetCheckSum()为任何OutputStream产生校验和（不仅是压缩）
    DeflaterOutputStream    压缩类的基类
    ZipOutputStream         一个DeflaterOutputStream，用于将数据压缩成zip文件格式
    GZIPOutputStream        一个DeflaterOutputStream，用于将数据压缩成gzip格式
    InflaterInputStream     解压缩的基类
    ZipInputStream一个      一个InflaterInputStream，用于解压缩Zip文件格式的数据
    GZIPInputStream         一个InflaterInputStream，用于解压缩Gzip文件格式的数据
